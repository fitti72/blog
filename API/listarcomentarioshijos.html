BLOG

GET comentario/listar/:id

Direccion URL

http://localhost:9000/blog/comentario/listar/:id

Descripcion

Se encarga de listar todos los comentarios que se realizaron a un comentario en especifico, es decir sus hijos.

Parametro

Id del Comentario

Entradas XML

No posee XML

Salidas XML

<list>
<models.OD.ComentarioOD>
<id__c>2</id__c>
<texto>mi segundo comentario</texto>
<fecha>Thu May 31 19:55:52 VET 2012</fecha>
<privacidad>1</privacidad>
<id__u>2</id__u>
<padre>0</padre>
<gusta>0</gusta>
<nogusta>0</nogusta>
</models.OD.ComentarioOD>
</list>

Errores

<mensaje>Error: Lista vacia</mensaje>
