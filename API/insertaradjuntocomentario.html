BLOG

POST comentario/insertarAdjunto/:token

Direccion URL

http://localhost:9000/blog/comentario/insertarAdjunto/:token

Descripcion

Se encarga de insertar el adjunto del comentario en persistencia pasandole como parametro el token del usuario y por XML el mismo XML de el insertar comentario pero agregandole la etiqueta de la ruta.

Parametro

Token del usario

Entradas XML

Comentario Padre con Adjunto

<comentario>
<texto>comentario 2</texto>
<privacidad>1</privacidad>
<id_u>1</id_u>
<padre>0</padre>
<tags>
<tag1>etiqueta1</tag1>
<tag2>etiqueta2</tag2>
<tag3>etiqueta3</tag3>
</tags>
<ruta>rutaProvisionalCriminal</ruta>
</comentario>

 

Comentario hijo

<comentario>
<texto>comentario 2</texto>
<privacidad>1</privacidad>
<id_u>1</id_u>
<padre>0</padre>
<ruta>rutaProvisionalCriminal</ruta>
</comentario>

Salidas XML

<models.OD.ComentarioOD>
<id__c>2</id__c>
<texto>mi segundo comentario</texto>
<fecha>Thu May 31 19:55:52 VET 2012</fecha>

<id__u>2</id__u>
<padre>0</padre>
<gusta>0</gusta>
<nogusta>0</nogusta>
</models.OD.ComentarioOD>
