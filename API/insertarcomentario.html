BLOG

POST comentario/insertar/:token

Direccion URL

http://localhost:9000/blog/comentario/insertar/:token

Descripcion

Retorna el comentario ya insertado en persistencia con su id, en formato XML, se pasa como parametro el token del usuario que esta haciendo el comentario y un XML con el texto del comentario, la privacidad del comentario, el id del usuario creador el comentario, el padre, es decir, si esta respondiendo a un comentario ya realizado antes o un nuevo comentario y se le puede pasar de una vez los tags al comentario si es padre si no, no se le pasa tags.

Parametro

Token del Usuario

Entradas XML

Cuando es un comentario padre este es el formato de XML

<comentario>
<texto>mi segundo comentario</texto>
<privacidad>1</privacidad>
<id_u>2</id_u>
<padre>0</padre>
<tag>
<tag1>segundoTAG</tag1>
<tag2>quintoTAG</tag2>
<tag3>nuevoTag</tag3>
</tag>
</comentario>

Comentario Hijo

<comentario>
<texto>mi segundo comentario</texto>
<privacidad>1</privacidad>
<id_u>2</id_u>
<padre>0</padre>
</comentario>
