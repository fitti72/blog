BLOG

GET comentario/listar/tag/:tag

Direccion URL

http://localhost:9000/blog/comentario/listar/tag/:tag

Descripcion

Se encarga de listar todos los comentarios que esten asociados a un tags especifico se le pasa como parametro el tag.

Parametro

Nombre de TAG del usuario

Entradas XML

No posee XML

Salidas XML

<list>
<models.OD.ComentarioOD>
<id__c>2</id__c>
<texto>mi segundo comentario</texto>
<fecha>Wed May 30 08:06:39 VET 2012</fecha>
<privacidad>1</privacidad>
<id__u>1</id__u>
<padre>0</padre>
<gusta>1</gusta>
<nogusta>0</nogusta>
</models.OD.ComentarioOD>
</list>

Errores

<mensaje>Error: Tag no existe</mensaje>
