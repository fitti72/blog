BLOG

GET comentario/tags/:id_c

Direccion URL

http://localhost:9000/blog/comentario/tags/:id_c

Descripcion

Se encarga de listar todos los tags que esten asociados a un comentario, se le pasa por parametro el id del comentario

Parametro

Id del comentario

Entradas XML

No posee XML

Salidas XML

<list>
<string>segundotag</string>
<string>quintotag</string>
<string>nuevotag</string>
</list>

Errores

<mensaje>Error: Lista vacia</mensaje>
