BLOG

DELETE /comentario/eliminar/:nick/:id_c/:token

Direccion URL

http://localhost:9000/blog/comentario/:nick/:id_c/:token

Descripcion

Retorna un mensaje diciendo que se elimino el mensaje, ha este metodo se le pasa como parametro el nick del usuario, el id del comentario, y el token del usuario. este metodo elimina los comentarios que sean del usuario o si el usuario es dueno del comentario padre puede eliminar cualquier hijo a ese comentario.

Parametro

Token del Usuario

Nick del Usuario

Id del Comentario

Entradas XML

No posee XML

Salidas XML

<mensaje>elimine comentarioo</mensaje>

Errores

<mensaje>Error: Id del comentario vacio no esta permitido </mensaje>
<mensaje>Error: Indique el id del usuario </mensaje>
<mensaje>Error: El id del comentario debe ser un numero </mensaje>
<mensaje>Error: El token debe ser un numero </mensaje>
<mensaje>Error:Usuario no existe </mensaje>
<mensaje>Error: Ese token no pertenece a ese usuario </mensaje>
<mensaje>Error: Error al eliminar el mensaje </mensaje>
