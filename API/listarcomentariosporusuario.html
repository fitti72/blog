BLOG

GET comentario/listaru/:nick

Direccion URL

http://localhost:9000/blog/comentario/listaru/nick

Descripcion

Se encarga de listar todos los comentarios que se realizaron por un usuario en especifico. se le pasa como parametro el id del usuasio, si la liste viene vacia se impirme vacia. Si la lista viene vacia se imprime NULL

Parametro

Nick del usuario

Entradas XML

No posee XML

Salidas XML

<list>
<models.OD.ComentarioOD>
<id__c>1</id__c>
<texto>Segundo intento de prueba unitaria</texto>
<fecha>Mon May 28 13:20:33 VET 2012</fecha>
<privacidad>0</privacidad>
<id__u>1</id__u>
<padre>0</padre>
<gusta>0</gusta>
<nogusta>0</nogusta>
</models.OD.ComentarioOD>
</list>

Errores

<mensaje>Error: Lista vacia</mensaje>
