BLOG

POST comentario/like/:token

Direccion URL

http://localhost:9000/blog/comentario/like/:token

Descripcion

Retorna el comentario en XML al cual se le ha hecho la peticion de like o dislike, se le pasa como parametro el token del usuario el cual realixa la peticion, en el XML se le pasa el id del comentario, el id del usuario y la peticon de like o dislike.

Parametro

Token del Usuario

Entradas XML

<comentario>
<id_c>2</id_c>
<id_u>2</id_u>
<like>1</like>
</comentario>

Salidas XML

<mensaje>Registrado tu punteado al comentario</mensaje>

 

Errores

<mensaje>Error: Id del comentario vacio no esta permitido </mensaje>
<mensaje>Error: Indique el id del usuario </mensaje>
<mensaje>Error: Indique like=1 si le gusta, 0 si no le gusta </mensaje>
<mensaje>Error: El id del comentario debe ser un numero </mensaje>
<mensaje>Error: El id del usuario debe ser un numero </mensaje>
<mensaje>Error: El like debe ser un numero </mensaje>
<mensaje>Error: El like debe ser un numero de un solo digito </mensaje>
<mensaje>Error: El like solo puede ser 0 o 1 </mensaje>
<mensaje>Error:Ese token no pertenece a ese usuario </mensaje>
<mensaje>Error: no se puedo puntear el comentario</mensaje>
