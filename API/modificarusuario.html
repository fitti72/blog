BLOG

POST usuario/modificar/:token

Direccion URL

http://localhost:9000/blog/usuario/modificar/:token

Descripcion

Se encarga de modificar los datos del usuario pasandole un XML y el token del usuario como parametro para realizar la peticon y asi actualizar los datos del usuario.

Parametro

Token del usuario

Entradas XML

<usuario>
<id__u>8</id__u>
<nombre>Lucia</nombre>
<nombres>karina</nombres>
<apellido>Cardoso</apellido>
<apellidos>Milito</apellidos>
<email>lucia@hotmail.com</email>
<nick>lucia2</nick>
<pais>Portugal</pais>
<biografia>Desarrollando</biografia>
<sexo>M</sexo>
<fecha>21/06/1987</fecha>
<clave>1234</clave>
</usuario>
