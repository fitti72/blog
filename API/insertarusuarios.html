BLOG

POST usuario/insertar

Direccion URL

http://localhost:9000/blog/usuario/insertar

Descripcion

Se encarga de insertar en el sitema al usuario pasando un XML. Si no se pasa un XML el framework lanza un erro HTTP 400 ya que la funcion por defecto siempre esperara un XML.

Entradas XML

<usuario>
<nombre>Lucia</nombre>
<nombres>Ana</nombres>
<apellido>Cardosso</apellido>
<apellidos>Militao</apellidos>
<email>lucia@hotmail.com</email>
<nick>lucia2</nick>
<pais>Venezuela</pais>
<biografia>Desarrollando</biografia>
<sexo>M</sexo>
<fecha>21/06/1987</fecha>
<clave>1234</clave>
</usuario>
