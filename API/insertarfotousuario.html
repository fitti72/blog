BLOG

POST usuario/foto/:token

Direccion URL

http://localhost:9000/blog/usuario/foto/:token

Descripcion

Se encarga de registrar su foto de perfil en el sistema, pasandole por parametro el token y el XML del id del usuario y el nombre de la foto.

Parametro

Nick del usario

Entradas XML

<foto>
<id_u>1</id_u>
<ruta>rutaProvisional</ruta>
</foto>

Salidas XML

<mensaje>Se inserto bien la foto</mensaje>

Errores

<mensaje>Error: no se pudo insertar la foto </mensaje>
<mensaje>Error: Ese token no pertenece a ese usuario </mensaje>
