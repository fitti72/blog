GET usuario/datos/:id

Direccion URL

http://localhost:9000/blog/usuario/datos/:id

Descripcion

Retorna el nick de un usuario especifico se le pasa como parametro el id del usuario en formato XML.

Parametro

Id del Usuario

Entradas XML

No posee XML

Salidas XML
